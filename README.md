# Lox Interpreter in Rust

This is an implementation of the Lox programming language in Rust. The Lox programming language is
defined in the book ["Crafting Interpreters"] by Robert Nystrom.

The goal of this project is to learn about programming languages, interpreters and compilers.

The implementation of a Lox interpreter in Java and C implemented by Robert Nystrom during writing
the book can be found in his [repository "craftinginterpreters"].

["Crafting Interpreters"]: https://craftinginterpreters.com/

[repository "craftinginterpreters"]: https://github.com/munificent/craftinginterpreters
